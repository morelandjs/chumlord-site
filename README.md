# chumlord-site

Public landing page for [Chumlord](https://github.com/morelandjs/chumlord) — a
minimalistic friend-catchup tracker for iOS.

Hosted via GitHub Pages from the `main` branch root:
**https://morelandjs.github.io/chumlord-site/**

## Layout

```
index.html         # landing page
privacy.html       # privacy policy (canonical)
styles.css         # shared styles
PRIVACY.md         # markdown copy of the privacy policy
screenshots/       # iPhone screenshots used in the page
.nojekyll          # disable Jekyll processing on Pages
```

## Local preview

```sh
python3 -m http.server 8000
# → http://localhost:8000
```
